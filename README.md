# vmagueta.run 🚀

This is the source code for my personal engineering blog, hosted at [vmagueta.run](https://vmagueta.run).

The goal of this project is to document my journey through **Data Engineering**, **Cloud Infrastructure**, and **Software Development**, sharing technical challenges and solutions along the way.

## 🛠 Tech Stack

I chose a minimalist and high-performance stack to keep the focus on content and speed:

* **SSG:** [Marmite](https://marmite.run) – A simple, no-nonsense static site generator.
* **Content:** Written in **Markdown**.
* **Deployment:** Automated via **GitHub Actions**.
* **Hosting:** GitHub Pages.

## 🏗 Project Structure

* `/input`: Where the magic happens (Markdown files for posts).
* `/static`: Static assets like images and custom CSS.
* `marmite.yaml`: Configuration for the site generation.

## 🚀 How to build locally

If you want to run this project locally, you'll need the Marmite binary installed:

1. Clone the repo: `git clone https://github.com/vmagueta/vmagueta.run.git`
2. Run the server: `marmite serve input dist`
3. Open `localhost:8080` in your browser.
